# ChargeTrack

Система мониторинга напряжения аккумуляторов с уведомлениями в Telegram.

## Структура проекта

- `/backend`: FastAPI сервер и Telegram бот (aiogram).
- `/frontend`: Панель управления (админка) на React.

## Требования

- Python 3.9+
- Node.js 18+ (для фронтенда)

## Быстрый старт

### 1. Запуск Backend

1. Перейдите в папку `backend`:
   ```bash
   cd backend
   ```
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   venv\Scripts\activate     # Для Windows
   ```
3. Установите **PostgreSQL** и создайте базу данных `chargetrack`.
4. Настройте `.env` в папке `backend`:
   ```env
   BOT_TOKEN=ваш_токен
   DB_USER=postgres
   DB_PASSWORD=postgres
   DB_HOST=localhost
   DB_PORT=5432
   DB_NAME=chargetrack
   ```
5. Установите зависимости:
   ```bash
   pip install fastapi uvicorn sqlalchemy aiogram pydantic python-dotenv psycopg2-binary matplotlib apscheduler pytz
   ```
6. Запустите API сервер (для доступа из локальной сети используйте `--host 0.0.0.0`):
   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000 --reload
   ```
7. В другом терминале запустите Telegram бота:
   ```bash
   python bot.py
   ```
8. Запустите сервер приема данных от датчиков (если есть реальные данные):
   ```bash
   python serv.py
   ```

### 2. Запуск Frontend

1. Перейдите в папку `frontend`:
   ```bash
   cd frontend
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки (для доступа из локальной сети используйте `--host`):
   ```bash
   npm run dev -- --host
   ```
   *Примечание: двойное тире `--` нужно для передачи аргумента во внутренний скрипт vite.*

## Использование

1. Зайдите в панель управления:
    - Локально: `http://localhost:5173`
    - Из сети: `http://<IP_вашего_компьютера>:5173`
2. Авторизуйтесь под данными `adm1n:Q1092387456q`.
3. Добавьте нового подписчика, указав его имя и ID оборудования.
4. Скопируйте сгенерированный секретный код.
5. Передайте код пользователю. Он должен отправить его в Telegram бот.
6. После этого пользователь сможет использовать кнопки в боте:
    - **📊 Мониторинг**: получить текущее напряжение банок.
    - **📈 График**: получить график напряжений за последние 24 часа.
    - **🌅 Утренний отчет**: бот автоматически присылает график каждое утро в 08:00.

## Особенности демо-режима

Так как реальный сервер с датчиками пока не подключен, система генерирует случайные (но реалистичные) данные для демонстрации работы.
